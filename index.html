Update Content Method
üéôÔ∏è Narration
This method handles the actual text switching. It finds every element with both data-en and data-es attributes. For each element, it reads the appropriate data attribute based on selected language and updates the text. querySelectorAll finds all matching elements at once.
language-controller.js - Part 3
``` updateContent(lang) { const elements = document.querySelectorAll(‚Äô[data-en][data-es]‚Äô); ``` elements.forEach(el => { const content = el.getAttribute(`data-${lang}`); if (content) { if (el.tagName === 'INPUT' || el.tagName === 'TEXTAREA') { el.placeholder = content; } else { el.textContent = content; } } }); ``` }
```
Special Handling: Input and textarea elements need their placeholder attribute updated instead of textContent.

Page Language Attribute
üéôÔ∏è Narration
This small but important method updates the lang attribute on the HTML element. It tells browsers and screen readers what language the page uses. Screen readers use this to pronounce words correctly. Search engines use it to understand content better.
language-controller.js - Part 5
``` updatePageLang(lang) { document.documentElement.lang = lang; }
```
Accessibility Matters: Screen readers rely on the lang attribute to use correct pronunciation. This one line makes your site more accessible.

Performance Tips
üéôÔ∏è Narration
Optimize for larger sites. Cache DOM queries so you're not searching the entire document every time. Use event delegation for language buttons. Minify your JavaScript for production. These optimizations matter more as your site grows.
Performance Optimizations
``` class LanguageController { constructor() { this.currentLang = localStorage.getItem(‚Äòlanguage‚Äô) || ‚Äòen‚Äô; this.translatableElements = null; this.languageButtons = null; this.init(); } ``` cacheElements() { if (!this.translatableElements) { this.translatableElements = document.querySelectorAll('[data-en][data-es]'); } } updateContent(lang) { this.cacheElements(); this.translatableElements.forEach(el => { const content = el.getAttribute(`data-${lang}`); if (content) el.textContent = content; }); } ``` }
```
Performance Boost: Caching DOM queries can improve performance by 50% or more on pages with many translatable elements.
