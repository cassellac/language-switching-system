# Language Switching System

An interactive training module demonstrating how Digital Allies implements bilingual language switching on websites.

## Overview

This project provides an educational slideshow that explains how websites can seamlessly switch between English and Spanish without page reloads. It's designed to help team members and clients understand the technical implementation and benefits of the language switching system.

## Features

- **Interactive Training Slides**: 8 comprehensive slides covering all aspects of language switching
- **Live Demo**: Working language switcher demonstration within the training
- **Visual Learning**: Diagrams, code examples, and animations to illustrate concepts
- **Progressive Disclosure**: Step-by-step walkthrough from basics to technical details

## Topics Covered

1. **What Is Language Switching?** - Introduction to instant language switching
2. **How Does It Work?** - Three-step process explanation
3. **The Technical Side** - Data attributes and implementation details
4. **What Gets Translated?** - Content types and considerations
5. **The System Remembers You** - LocalStorage and user preferences
6. **Built For Everyone** - Accessibility features
7. **Fast and Efficient** - Performance characteristics
8. **Key Takeaways** - Summary and next steps

## Usage

Simply open `lang-switch-training.html` in any web browser to start the interactive training. Use the navigation buttons to move through the slides at your own pace.

## Technical Details

The system uses:
- **Data Attributes**: Stores translations in `data-en` and `data-es` attributes
- **LocalStorage**: Persists user language preference across sessions
- **Vanilla JavaScript**: No frameworks required
- **Responsive Design**: Works on all device sizes

## Browser Compatibility

Works in all modern browsers (Chrome, Firefox, Safari, Edge). No special setup or dependencies required.
